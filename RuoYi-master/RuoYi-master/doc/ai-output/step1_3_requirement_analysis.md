## ③ 业务流程

### 项目生命周期
```
创建项目(筹备中) → 启动项目(进行中) → 创建并分配任务 → 所有任务完成 → 关闭项目(已完成)
                                                          ↓
                                                    项目暂停(已搁置)
```

### 任务生命周期
```
创建任务(待处理) → 分配给成员 → 成员开始执行(进行中) → 提交完成(已完成) → 验收关闭(已关闭)
                                    ↓                       ↓
                              任务超期(系统标红)        验收不通过(退回进行中)
```

### 异常流程
- **删除项目**：如果项目下有未完成的任务，提示"请先完成或删除所有任务"
- **删除任务**：仅"待处理"状态的任务可以删除
- **任务超期**：超过截止日期的任务在列表中标红显示
- **权限越界**：普通成员不能修改他人的任务状态
